Random Name Selector

This selector will take in a list of names and generate a number of random names according to user input.
Please enter names or other words in the below input field. Each word should be seperated by a '/'. FOr example, to choose a random name between Bob, Billy, and Joe, I would enter 'Bob/Billy/Joe' into the input field. You can also choose how many random names to select.
